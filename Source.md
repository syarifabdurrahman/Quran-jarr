quranapi.pages.dev/api/

this for get chapter or surah:
Make a GET request to /<surahNo>.json

this for get verse
/<surahNo>/<ayahNo>.json

this for get tafsir ayah
Request Format
Make a GET request to api/tafsir/<surahNo>_<ayahNo>.json

the translation
Make a GET request to /<translationName>.json

english - English Translation
arabic1 - Arabic Translation with Tashkeel (diacritics)
arabic2 - Arabic Translation without Tashkeel (diacritics)
bengali - Bengali Translation
urdu - Urdu Translation
turkish - Trukish Translation
uzbek - Uzbek Translation

get reciters:
Make a GET request to /reciters.json

audio recitation:
Make a GET request to /audio/<surahNo>/<ayahNo>.json
